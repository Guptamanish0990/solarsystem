# Solar System Website Project

## Overview
This Solar System website project is designed to provide users with an interactive and educational experience about the solar system. It showcases various celestial bodies within our solar system, including planets, moons, asteroids, and more.

## Features
- **Interactive Interface**: Users can explore the solar system through an engaging and user-friendly interface.
- **Educational Content**: Each celestial body has detailed information, including size, distance from the sun, and interesting facts.
- **Visuals**: High-quality images and diagrams to enhance user understanding and interest.
- **Responsiveness**: The website is designed to be fully responsive, ensuring accessibility on various devices.

## Technologies Used
- **HTML**: For the basic structure of the web pages.
- **CSS**: For styling the website and ensuring a good user interface. 
- **JavaScript**: For adding interactivity to the site.
