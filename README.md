# Solar System Explorer

This project is an **interactive 3D web application** that allows users to explore a simulation of our solar system. By clicking on different celestial bodies, users can access a wealth of information about them in an engaging, visual way.

You can view the live site here: **[https://guptamanish0990.github.io/solarsystem/](https://guptamanish0990.github.io/solarsystem/)**

## ✨ Key Features

*   **Live 3D Simulation:** A dynamic, visually rich model of the solar system where planets orbit the Sun.
*   **Interactive Exploration:** Simply click on any planet or celestial body to select it and learn more.
*   **Comprehensive Information Panels:** Once a body is selected, detailed categories of information become available, including:
    *   **General Info:** Key facts and figures about the selected body.
    *   **Surface:** Details about its surface composition and features.
    *   **Missions:** A list of historical or planned space missions to that body.
    *   **Satellites:** Information about its moons (if any) or ring systems.
    *   **Minerals:** Details on the known mineral composition or geological elements.
*   **Simulation Controls:** You can control the simulation speed or pause the orbits for closer observation.

## 🎯 Purpose

The goal is to make learning about our solar system immersive and fun. This tool serves as an educational resource for astronomy enthusiasts, students, or anyone curious about space, allowing them to visualize planets and access key data interactively.

## 🚀 How to Use

1.  Open the website link.
2.  Observe the live simulation of the solar system.
3.  Click on any planet or the Sun to select it.
4.  Use the information tabs (Info, Surface, Missions, etc.) that appear on the screen to explore details.
5.  Use the **Speed** and **Pause** buttons to control the simulation's pace.
